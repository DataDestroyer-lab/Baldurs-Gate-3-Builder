# Baldurs-Gate-3-Builder# Baldur's Gate 3 Builder - Wiki Accurate Edition

**A comprehensive, single-file HTML character builder for Baldur's Gate 3.**

This tool allows players to plan their Level 1 characters with 100% wiki-accurate data, including all races, subraces, classes, subclasses, backgrounds, and point-buy stats. It features a high-contrast, large-print UI for accessibility and includes an embedded music player for immersion.

* **Wiki-Accurate Data:** Includes all launch-day races, subraces, classes, and subclasses.
* **Point Buy Calculator:** Fully functional 27-point buy system with racial bonuses (+2/+1).
* **Skill Manager:** Smart skill selection that prevents duplicate picks from Backgrounds/Races.
* **Level 1 Options:** Select Fighting Styles, Deities, Draconic Ancestries, and Sorcerous Origins.
* **Spell Picker:** Select starting Cantrips and Level 1 Spells for all caster classes.
* **Large Print UI:** Designed with accessibility in mind‚Äîlarge text, high contrast, and big click targets.
* **Immersive Audio:** Integrated YouTube player featuring "Down By The River" (Official OST).
* **Single File:** The entire app is contained in one `index.html` file. No servers or installation required.
